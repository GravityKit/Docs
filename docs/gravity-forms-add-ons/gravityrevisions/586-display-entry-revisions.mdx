# How to display entry revisions on your website

The [GravityRevisions](https://www.gravitykit.com/extensions/entry-revisions/) plugin allows you to display entry revisions within a View on the front end of your website.

We will use [GravityRevisions Merge Tags](https://docs.gravitykit.com/article/478-entry-revision-merge-tags) and a [Custom Content field](https://docs.gravitykit.com/article/111-using-the-custom-content-field) to show changes to entries. To get started, edit your View then:

### Click the "Add Field" button

![A mouse clicking the "Add Field" button](https://www.gravitykit.com/wp-content/uploads/2018/01/click-add-field-.png?1432232558)

### Scroll down and click on the Custom Content field

![A pop-up showing a list of available fields. The mouse is hovering over "Custom Content".](https://www.gravitykit.com/wp-content/uploads/2018/01/scroll-down-and-click-on-the-custom-content-field.png?1432232559)

### Click on the gear icon to configure the Custom Content field

![Clicking the gear icon to configure the Custom Content field](https://www.gravitykit.com/wp-content/uploads/2018/01/click-on-the-field-settings-gear-to-customize-it.png?1432232559)

### Add an Entry Revisions Merge Tag in the Custom Content box

Click the Merge Tags icon to the right of the Custom Content area to show a dropdown with all available merge tags. In this example, we selected the Entry Revisions List tag, one of the available [GravityRevisions Merge tags](https://docs.gravitykit.com/article/478-entry-revision-merge-tags), which will display a list of changes made to an entry over time.

![Using the Merge Tags dropdown menu to select the Entry Revisions List merge tag](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d0bd9750428637fd7c53bad/file-W3MVCABF7t.png)

### Save or Update your View

![Clicking the Update button to save the modified View](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d0bde900428637fd7c53c0c/file-XD4ISXQkvK.png)

### Revisions are now visible on the front-end

Each change displays a thumbnail-sized image of the user who made the change, the date the change was made, and a link to restore the entry values from that change.

![A list of revisions to the entry are now visible on the website](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d0bdf202c7d3a6b51c6af3b/file-JtlpZbv1yg.png)
