# Field types not yet supported by GravityActions

While GravityActions supports most of the fields in Gravity Forms, some more complex ones are not yet supported.

Here's the list of **non-supported** fields and entry meta:

### Advanced Fields

- List
- Consent
- File Upload
- HTML Block
- Multi-Select

### Post Fields

- Post Title
- Post Body
- Post Excerpt
- Post Tags
- Post Category
- Post Image
- Custom Fields

### Pricing Fields

- Product
- Quantity
- Product Option
- Shipping
- Total

### Entry Meta 

- `currency`
- `payment_status`
- `payment_date`
- `payment_amount`
- `payment_method`
- `transaction_id`
- `transaction_type`
