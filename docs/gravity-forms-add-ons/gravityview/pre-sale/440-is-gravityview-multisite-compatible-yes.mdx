# Is GravityView Multisite compatible? Yes!

Yes, GravityView is [Multisite](https://wordpress.org/support/article/create-a-network/) compatible! GravityView works well whether the plugin is network-activated or site-activated.

For information about GravityView and licensing a Multisite network, [read this article](https://docs.gravitykit.com/article/439-multisite-and-gravityview-licenses).

If you have questions about setting up WordPress Multisite, [read this article about Multisite](https://wpdatatables.com/wordpress-multisite/). It's very informative!
