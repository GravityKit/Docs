# Turning off the Show Search Bar default search field



### Click the Multiple Entries tab

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-multiple-entries-tab.png?1417725511)

### Click the gear icon next to the Search Bar widget

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-gear-icon-next-to-the-search-bar-widget.png?1417725512)

### Click the "X" icon next to the Search Everything option

This removes the search field that searches all entries.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-x-icon-next-to-the-search-everything-option.png?1417725513)

### Click "Update" to save your View

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-update-to-save-your-view.png?1417725514)
