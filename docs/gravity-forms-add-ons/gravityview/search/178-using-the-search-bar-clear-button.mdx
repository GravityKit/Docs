# Using the Search Bar "Clear" button

**Added in Version 1.5.4**

The Search bar widget has a "Show Clear button" setting. Enabling this setting allows users to clear entered search values.

## Click the gear to open the Search Bar widget settings

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-gear-to-open-the-search-bar-widget-settings.png?1422596332)

## Check the Show Clear button checkbox

After checking the box, you can close the Search Bar settings window.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/check-the-show-clear-button-checkbox.png?1422596333)

## Save the View

![](https://www.gravitykit.com/wp-content/uploads/2018/01/save-the-view.png?1422596333)

## Click "View on website" to see the View

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-view-on-website-to-see-the-view.png?1422596334)

## You'll not see anything different at first....

![](https://www.gravitykit.com/wp-content/uploads/2018/01/you-ll-not-see-anything-different-at-first.png?1422596334)

## ...until you start typing!

When you start typing, the Reset button will appear. When clicking the Reset button, the form will revert to its original state.

If you have performed a search and the form has existing values, clicking Reset will return the original values to the form.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/until-you-start-typing-.png?1422596335)

## If you click Reset…

![](https://www.gravitykit.com/wp-content/uploads/2018/01/if-you-click-reset-hellip-.png?1422596338)

## …it'll return to its original state.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/hellip-it-ll-return-to-its-original-state.png?1422596341)

## If there's an existing search value, "Clear" will be visible on load

Clicking clear will remove all search values.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/if-there-s-an-existing-search-value-clear-will-be-visible-on-load.png?1422596343)
