# Adding custom CSS to your website



### Click on Appearance, then Customize

![The Customize link under the Appearance menu item in the WordPress Admin menu](https://www.gravitykit.com/wp-content/uploads/2018/02/click-on-appearance-then-customize.png)

### Click the Additional CSS menu item at the bottom of the side menu.

![The Additional CSS menu item in the WordPress Customizer panel](https://www.gravitykit.com/wp-content/uploads/2018/02/click-the-additional-css-menu-item-at-the-bottom-of-the-side-menu.png)

### Add your CSS code in the editor and click "Update Custom CSS"

Add your CSS code to the text editor. The CSS you enter will be live-previewed for you. 

![CSS code inside the CSS editor](https://www.gravitykit.com/wp-content/uploads/2018/02/add-your-css-code-in-the-editor-and-click-update-custom-css-.png)

### Click the Publish button at the top of the sidebar

![The Publish button at the top of the Customizer](https://www.gravitykit.com/wp-content/uploads/2018/02/click-the-publish-button-at-the-top-of-the-sidebar.png)

Once you click "Publish", the changes will go live. You can also click the gear icon to save your CSS as a draft or schedule the changes you would like to make. 

---

### If you're running a WordPress version lower than 4.9

First: you should update your site! Running the latest WordPress is the best way to prevent your site from being hacked. Now you've been warned :-)

If you're running under WordPress 4.9, follow the steps below: Note: if you already have the [Jetpack plugin](https://wordpress.org/plugins/jetpack) enabled, you can use their [Custom CSS module](http://jetpack.me/support/custom-css/) instead. 

#### Click on Plugins, then Add New

![The Add New link underneath the Plugins menu item in the WordPress Admin menu](https://www.gravitykit.com/wp-content/uploads/2018/02/click-on-plugins-then-add-new.png)

#### Enter "Simple Custom CSS" into the search field

This is the name of the plugin we're going to use to add custom CSS. 

![Searching for a plugin called Simple Custom CSS](https://www.gravitykit.com/wp-content/uploads/2018/02/enter-simple-custom-css-into-the-search-field.png)

#### Click the "Install Now" button

![The Install Now button for the Simple Custom CSS plugin](https://www.gravitykit.com/wp-content/uploads/2018/02/click-the-install-now-button.png)

#### Activate the plugin

![The Activate Plugin link](https://www.gravitykit.com/wp-content/uploads/2018/02/activate-the-plugin.png)

#### Click on the "Custom CSS" link under Appearance

![The Custom CSS link under the Appearance menu item in the WordPress Admin menu](https://www.gravitykit.com/wp-content/uploads/2018/02/click-on-the-custom-css-link-under-appearance.png)

#### Add your CSS code in the editor and click "Update Custom CSS"

Delete the line that says "/* Enter Your Custom CSS Here */", then add your CSS code in the text editor. Once you're done, click "Update Custom CSS." 

![The Update Custom CSS button below the CSS editor](https://www.gravitykit.com/wp-content/uploads/2018/02/add-your-css-code-in-the-editor-and-click-update-custom-css-1.png)
