# Adding Custom CSS classes to fields or widgets


:::caution

Note: Custom CSS is always enabled for *fields.* To enable custom CSS classes for GravityView *widgets*, [first follow this how-to guide.](https://docs.gravitykit.com/article/179-enabling-custom-css-classes-for-gravityview-widgets)

:::



### Click the gear icon to access the field or widget settings

The gear icon will appear when configurations are available for the field or the widget.

![Screenshot of the View editor](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6063b341d97e2f20e67cf459/file-ScZ1NSsRCT.png)

### Add your custom CSS classes

You can add CSS classes that will be added to the field container when rendered in the View.

GravityView provides some classes for modifying widths. [Learn more on the CSS Guide's](https://docs.gravitykit.com/article/63-css-guide) "Grid Layout" section.

![Screenshot of the field settings panel](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6063b38724ce107ab4d1b173/file-w8yyTFPazD.png)
