# Changing column width



### Sometimes a column just isn't the right size!

In this example, we are going to adjust the width of the Summary column. Don't you think it's too wide? We do too.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/sometimes-a-column-just-isn-t-the-right-size-.png?1435923183)

### Edit the View you want to modify

You can normally access it from the Admin Bar if you're currently on the View page.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/edit-the-view-you-want-to-modify.png?1435923186)

### In View Configuration, click the Field gear icon for the field you wish to modify

![](https://www.gravitykit.com/wp-content/uploads/2018/01/in-view-configuration-click-the-field-gear-icon-for-the-field-you-wish-to-modify.png?1435923187)

### Enter the desired column width in %

Note: GravityView will not check to make sure that your widths stay up to 100%

![](https://www.gravitykit.com/wp-content/uploads/2018/01/enter-the-desired-column-width-in-.png?1435923188)

### Save the View

![](https://www.gravitykit.com/wp-content/uploads/2018/01/save-the-view.png?1435923190)

### Return to the View, and refresh the page

The field's column should now be the width specified.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/return-to-the-view-and-refresh-the-page.png?1435923191)
