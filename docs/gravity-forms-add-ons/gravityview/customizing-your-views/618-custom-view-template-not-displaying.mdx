# Custom single View template not displaying

If you're [overriding the GravityView template](https://docs.gravitykit.com/article/64-overriding-gravityview-templates) with a custom `single-gravityview.php` template and it's not displaying, double-check your theme's structure. If you're using a child theme, move your custom template file to the root folder of the child theme:

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d53002c2c7d3a68825e9ad6/file-xjpY0H6Sym.png)

Your custom View template should then start displaying.
