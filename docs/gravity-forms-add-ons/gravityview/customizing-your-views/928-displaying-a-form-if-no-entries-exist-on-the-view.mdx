# Displaying a form if no entries exist on the View

Since GravityView version 2.17, it is now possible to display a form if no entries are displayed on the View.

Inside the View editor, on the View Settings metabox &gt; Multiple Entries tab, you can define the behavior of the View when no entries exist:

![Screenshot showing the Multiple Entries tab inside the View Settings metabox](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/642b56fd91efbc3c6264a207/file-tpXavmHs07.png)

It is also possible to redirect users to an URL:

![Screenshot showing the Redirect to URL option on the View Settings metabox](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/642b5791a3236b1edbce8ee7/file-g4JHZ7l49k.png)
