# Removing column headings from the bottom of the tables

If you are using the Table or DataTables view type...

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/57c07105903360342852e5e3/file-BvqYeNTuuG.jpg)

...you probably want to hide the column headings from the bottom of those tables:

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/57c0718b903360342852e5ea/file-3Lw2R5YiXE.jpg)

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/57c071d5903360342852e5ef/file-uBaYQLSkJV.jpg)

Then, you just need to apply these CSS styles to your theme stylesheet ( [How to add custom CSS to your website](https://docs.gravitykit.com/article/246-adding-custom-css-to-your-website)):

For Table views:

```
.gv-table-view tfoot{ display:none; }
```

For DataTables:

```
.gv-datatables tfoot{ display: none;}
```
