# How does GravityView work with Gravity Forms "Save and Continue"?

### What is Save and Continue?

Gravity Forms allows you to save a form and continue filling it out later. This functionality is called "Save and Continue."


:::caution

⚠️ GravityView **does not** support displaying and editing Save and Continue entries.

:::

  GravityView does support [displaying partial Gravity Forms entries that were created using the Partial Entries Add-on](https://docs.gravitykit.com/article/466-save-and-continue-partial-entries).

#### Does GravityView display entries that have been saved?

No; GravityView will not display Saved entries; GravityView only displays completed form entries.

#### Is there a way to display saved entries?

No, GravityView does not support displaying saved entries.
