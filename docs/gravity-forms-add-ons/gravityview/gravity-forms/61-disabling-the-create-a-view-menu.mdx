# Disabling the Create a View menu

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/634732a162a58869ee5f1147/file-l8xTSAo1HO.png)

If you would like to remove the "Create a View" menu from the Gravity Forms form toolbar, [add this code to your site's functions.php file](https://docs.gravitykit.com/article/210-where-to-put-code-samples):

```
remove_filter( 'gform_toolbar_menu', array( 'GravityView_Admin_Views', 'gform_toolbar_menu' ), 10, 2 );
```
