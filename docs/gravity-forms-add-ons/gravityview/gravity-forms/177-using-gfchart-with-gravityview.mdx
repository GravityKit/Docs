# Using GFChart with GravityView


:::info

GravityKit now has its own chart plugin called [GravityCharts](https://www.gravitykit.com/extensions/gravitycharts/). Make sure to check it out!

:::



GFChart is an easy way to display your Gravity Forms entry data in a chart. GFChart does not currently support GravityView filters and sorting, but if you're looking for an easy way to display form results, [GFChart](https://gfchart.com/?ref=2) is an excellent option.

### Click Add Widget in the Edit View screen

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-add-widget-in-the-edit-view-screen.png?1422661000)

### Click the Custom Content widget

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-custom-content-widget.png?1422661001)

### Click the gear to configure the Custom Content settings

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-gear-to-configure-the-custom-content-settings.png?1422661002)

### Add the GFChart shortcode for your form

You can learn how to use the GF Chart shortcode [on the GFChart website](https://gfchart.com/?ref=2) (under the "Shortcode" heading).

![](https://www.gravitykit.com/wp-content/uploads/2018/01/add-the-gf-chart-shortcode-for-your-form.png?1422661003)

#### Finding the Field ID

If you aren't sure what field ID to use in the GFChart shortcode, click the "Add Field" button to display the field picker. There, you will see the field ID next to the name of each field.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/finding-the-field-id.png?1422661003)

### Save your View
