# There are other plugin boxes on the Edit View screen

Other plugins add configuration boxes (called <q>metaboxes</q>) to the Edit View screen. Here's how to disable them.

## Ugly configuration boxes getting you down?

![](https://www.gravitykit.com/wp-content/uploads/2018/01/ugly-configuration-boxes-getting-you-down-.png?1406699357)

## Click the <q>Screen Options</q> tab in the top-rigth of the page.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-screen-options-tab-in-the-top-rigth-of-the-page.png?1406699358)

## Uncheck the boxes you don't want to see

![](https://www.gravitykit.com/wp-content/uploads/2018/01/uncheck-the-boxes-you-don-t-want-to-see.png?1406699359)

## And click the Screen Options tab again.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/and-click-the-screen-options-tab-again.png?1406699360)

## And you're all set!

The best news? This tip works for all posts and page edit screens, not just Views. Check out the Screen Options tab, it's great!

![](https://www.gravitykit.com/wp-content/uploads/2018/01/and-you-re-all-set-.png?1406699361)
