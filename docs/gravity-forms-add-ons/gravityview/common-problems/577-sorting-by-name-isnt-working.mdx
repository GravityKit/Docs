# Sorting by Name isn't working

If sorting your View by "Name" isn't working, it's usually because the "Name" field on your form is set up as a Single Line Text field type. Sorting by first or last name requires the form to have a "Name" field, which is located under "Advanced Fields" on the Gravity Forms "Edit Form" screen:
