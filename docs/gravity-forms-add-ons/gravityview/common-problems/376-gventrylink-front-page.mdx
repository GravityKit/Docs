# Pointing the [gv_entry_link] to a specific page



### Here's a \[gv\_entry\_link\] shortcode in a Custom Content field

![](https://www.gravitykit.com/wp-content/uploads/2018/01/here-s-a-gv_entry_link-shortcode-in-a-custom-content-field.png?1472600558)

### The link points to the View, not to the page.

Even though the View is embedded on the home page, the link is pointing to the View.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/the-link-points-to-the-view-not-to-the-home-page.png?1472600558)

### So how to fix this?

We need to use the **post\_id** parameter on the \[gv\_entry\_link\] shortcode like this:

`[gv_entry_link post_id="#"]`, where the # is the ID of the page where your View is embedded.

### Getting the Page ID

If you know the page ID of the page, update the shortcode to use it. For example, if the page ID is 2, the shortcode would be `[gv_entry_link post_id="2"]`. **The link should work now.**

If you need help finding the page ID, read on!

#### Go to the Pages menu

Click on the name of the page in the "Pages &gt; All Pages" list.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-on-the-name-of-the-page-in-the-pages-gt-all-pages-list.png?1472600561)

#### The page ID will be in the URL as the "post" parameter

But this won't work for everyone because browsers don't always show the full URL.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/the-page-id-will-be-in-the-url-as-post-number-.png?1472600561)

#### Don't see the full URL and need to find the Page ID?

Go to Plugins &gt; Add New, then search for "Reveal IDs". Click the Install Now button, then click Activate to activate the plugin.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/don-t-see-the-full-url-and-need-to-find-the-page-id-.png?1472600562)

#### You will now see an ID column next to each page.

That number is the page ID we've been looking for. Sorry it's so hard to find!

![](https://www.gravitykit.com/wp-content/uploads/2018/01/you-will-now-see-an-id-column-next-to-each-page.png?1472600563)
