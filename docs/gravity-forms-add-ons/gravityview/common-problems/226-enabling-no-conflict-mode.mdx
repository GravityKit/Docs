# Enabling No-Conflict Mode



### Go to the GravityView Settings page

Hover over "Views" then click "Settings".

![](https://www.gravitykit.com/wp-content/uploads/2018/01/go-to-the-gravityview-settings-page.png?1432153465)

### Turn on No-Conflict Mode

Click "On" to turn on the No-Conflict Mode.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/turn-on-no-conflict-mode.png?1432153466)

### Click "Update Settings" to save the settings

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-update-settings-to-save-the-settings.png?1432153467)
