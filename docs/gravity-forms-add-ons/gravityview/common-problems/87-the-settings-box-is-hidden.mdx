# The Settings box is hidden



### 1. Click the "Screen Options" tab at the top-right of the page.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-screen-options-tab-in-the-top-rigth-of-the-page.png?1432159798)

### 2. Make sure Settings is checked

![Screenshot of the Screen Options box](https://i.gravitykit.com/sMQg3K+)

### 3. And click the Screen Options tab again

This will close the pane.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/and-click-the-screen-options-tab-again.png?1432159800)

### 4. And you're all set!

The Settings box should now be visible below the box with the View layouts.

![Screenshot of the Settings box](https://i.gravitykit.com/rCrQHH+)
