# How to delete and re-install GravityView

## Deactivate the GravityView plugin

![](https://www.gravitykit.com/wp-content/uploads/2018/01/deactivate-the-gravityview-plugin.png?1407261229)

## Once the plugin is de-activated, click the Delete link

![](https://www.gravitykit.com/wp-content/uploads/2018/01/once-the-plugin-is-de-activated-click-the-delete-link.png?1407261230)

## Click the "Yes, Delete these files and data" button

Don't worry about the warning text; your Views and settings will not be deleted.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-yes-delete-these-files-and-data-button.png?1407261231)

## Now re-install GravityView

Now that you have removed GravityView, follow the instructions on [How to Install GravityView](https://docs.gravitykit.com/article/69-installing-gravityview). Once you re-install it, your Views will still be there.
