# Visual Composer and GravityView



### Do you see Visual Composer buttons when you edit a View?

If you don't see the buttons, then it's another issue; [contact us](https://www.gravitykit.com/support/)!

![](https://www.gravitykit.com/wp-content/uploads/2018/01/do-you-see-visual-composer-buttons-when-you-edit-a-view-.png?1423023982)

### 1. Go to the Visual Composer settings

![](https://www.gravitykit.com/wp-content/uploads/2018/01/go-to-the-visual-composer-settings.png?1423023982)

### 2. Un-check "gravityview"

![](https://www.gravitykit.com/wp-content/uploads/2018/01/un-check-gravityview-.png?1423023983)

### 3. Click "Save Changes"

Make sure to save the Visual Composer settings.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-save-changes-.png?1423023984)

### 4. Editing your View should work again!
