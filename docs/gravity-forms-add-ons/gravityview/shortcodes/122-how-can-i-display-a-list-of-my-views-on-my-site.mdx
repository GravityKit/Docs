# How can I display a list of my Views on my site?

To display a list of Views, we recommend installing the [Display Posts Shortcode](https://wordpress.org/plugins/display-posts-shortcode/) plugin. Once installed, you can display a list of Views by using the following shortcode:

`[display-posts post_type="gravityview"]`
