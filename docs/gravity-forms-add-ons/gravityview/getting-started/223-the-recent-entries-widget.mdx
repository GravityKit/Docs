# The Recent Entries Widget



### Go to the Widgets page

![](https://www.gravitykit.com/wp-content/uploads/2018/01/go-to-the-widgets-page.png?1423806767)

### Click on the widget labeled "GravityView Recent Entries"

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-on-the-widget-labeled-gravityview-recent-entries-.png?1423806768)

### Click the sidebar you want to add the widget to

Sites have sidebars that display in different places in the theme. Choose the sidebar you wish to add the search form to, then click the "Add Widget" button.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-sidebar-you-want-to-add-the-widget-to.png?1423806770)

### This is what the Widget looks like initially

1. The widget title-this will be displayed above the search form
2. The View selector-You will pick what View you want to display entries from
3. Number of entries to show-Choose how many entries you would like to show at once
4. The text to use as the link to the single entry
5. Additional content to show below the link

![](https://www.gravitykit.com/wp-content/uploads/2018/01/this-is-what-the-widget-looks-like-initially.png?1423806770)

### Give the widget a title

This will be displayed as a heading above the entries.

In this example, we'll be creating using the widget to display the most recent events in an Event Listings View.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/give-the-widget-a-title.png?1423806771)

### Select a View

Select the View you want to search from the dropdown.

*If you haven't configured any Views, you'll need to do that first.*

![](https://www.gravitykit.com/wp-content/uploads/2018/01/select-a-view.png?1423806772)

### Once you select a View, Merge Tags will be visible

Clicking on the Merge Tag icon will present you with a list of fields from the form connected to the View. By inserting merge tags into the fields, you can create complex Custom Content ( [learn more about Custom Content](https://docs.gravitykit.com/article/111-using-the-custom-content-field)).

![](https://www.gravitykit.com/wp-content/uploads/2018/01/once-you-select-a-view-merge-tags-will-be-visible.png?1423806773)

### Once you're done configuring, click the "Save" button

![](https://www.gravitykit.com/wp-content/uploads/2018/01/once-you-re-done-configuring-click-the-save-button.png?1423806773)

### Our fully-configured Recent Entries widget

Note the merge tags used in the Link Text and the textarea below match up with the link text and content below the link in the next screenshot.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/our-fully-configured-recent-entries-widget.png?1423806774)

### And here's the configured Recent Entries widget

You can configure the Recent Entries widget to display the top results from your View and customize the layout.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/and-here-s-the-configured-recent-entries-widget.png?1423806775)
