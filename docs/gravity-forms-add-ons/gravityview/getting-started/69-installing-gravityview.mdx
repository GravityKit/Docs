# Installing GravityView


This article will walk you through how to install the GravityView plugin.

### Prefer to Watch the Video?

<iframe allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen="" frameborder="0" height="315" src="https://www.youtube.com/embed/73hl8R4Oc7M" title="YouTube video player" width="560"> </iframe>

### 1. Download the Latest Version of GravityView

Go to the [GravityKit website](https://www.gravitykit.com/account/), hover over "Account" in the top right and click "Log in". After logging in, click on the "[Downloads](https://www.gravitykit.com/account/#your-downloads)" tab.

![The different tabs on the GravityView Account page with the Download tab highlighted](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5ffcae69551e0c2853f3b332/file-IrvtPdOmCe.png)

Then, scroll down and click the "Download Extension" button to download the latest version of GravityView.

![The Download Extension button next to the GravityView plugin](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5ffcae46bb5c6f7434e0fb64/file-9nfKBGtiBR.png)

2. Install GravityView on Your WordPress Website 

Log in to your WordPress admin backend and click on "Plugins" in the left-hand menu. Next, click "Add New".

![The Add New button on the WordPress Plugins page](https://www.gravitykit.com/wp-content/uploads/2018/01/click-on-the-plugins-menu-in-wordpress-dashboard-then-click-the-add-new-button.png?1479253149)

Now click "Upload Plugin".

![The Upload Plugin button on the WordPress Plugins page](https://www.gravitykit.com/wp-content/uploads/2018/01/clcik-the-upload-plugin-button-at-the-top-of-the-add-plugins-page.png?1479253150)

Click "Choose File".

![The Choose File button on the WordPress Plugins page](https://www.gravitykit.com/wp-content/uploads/2018/01/click-choose-file-to-select-the-gravityview-plugin-file.png?1479253150)

Now select the GravityView plugin file that you downloaded in the previous step.

![Selecting the GravityView plugin ZIP file from a local folder](https://www.gravitykit.com/wp-content/uploads/2018/01/choose-the-downloaded-file-from-your-computer.png?1479253151)

Finally, click "Install Now".

![The Install Now button on the GravityView Plugin page](https://www.gravitykit.com/wp-content/uploads/2018/01/click-install-now-.png?1479253152)

After the plugin has finished installing, activate it by clicking "Activate Plugin".

![The Activate Plugin button](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-activate-plugin-button.png?1479253153)

You will now see the GravityView "Getting Started" screen with information on configuring your first View.

![The GravityView Getting Started page inside WordPress](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6081cbd63149d33a19c6f910/file-fABZf2FGla.png) Before you start creating new Views, you should first activate your license.

### 3. Activate Your GravityView License

After activating the GravityView plugin, hover over "Views" and click "Settings".

![The GravityView menu inside the WordPress admin panel](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6081ccb6f8c0ef2d98df53bf/file-6dxLpIEe9W.png)

Go back to your Account page on GravityView.co, [copy your license key](https://www.gravitykit.com/extensions/advanced-filter/), and paste it into the License Key box.

![The GravityView settings page with a blank license key input field](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/610b9813766e8844fc34dbd1/file-trT8COCYx6.png)

After entering your key, press the button that says "Activate License".

![Screenshot of the License Key input field containing a license key](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6081cf8e8af76a714bfda085/file-DJsI8zNfBH.png)

After activating your license, you should see a notice saying that the license key is valid and active, along with your account details below. If not, [contact our support team](https://www.gravitykit.com/support/).![The GravityView Settings page showing the License information](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6081d042f8c0ef2d98df53db/file-eVKbunoFRB.png)

After activating your license, click the "Save Settings" button.

![The Save Settings button](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6081d06ee0324b5fdfd0dc64/file-oUGIeQYlmj.png)

Now that your license is active, you can enjoy automatic updates!

If you ever have questions about the plugin, you can click on the blue circle with a question mark in the middle at the bottom of each GravityView page (we call that the "Support Port").

From there, you can also search our how-to articles and send support requests - all without leaving your website.

![A blue circle with a question mark inside](https://www.gravitykit.com/wp-content/uploads/2018/01/now-gravityview-has-been-activated-you-will-receive-automatic-updates.png?1479253159)
