# The View Settings Meta Box

### Prefer to Watch the Video?

<iframe allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen="" frameborder="0" height="315" src="https://www.youtube.com/embed/c3Ulu509PIU" title="YouTube video player" width="560"></iframe>

 Below the View editor, you'll see a Settings meta box that allows you to customize and modify different aspects of your View. Some of these settings are important for controlling the visibility of the entries or even the entire View. 

 ![The GravityView View Settings meta box below the View editor](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/610a57b2766e8844fc34d599/file-OIx8LQTNn4.png)


:::info

**Note**: As you can see, there are 7 tabs on the left-hand side that contain settings for different aspects of a View. These 7 tabs are the default settings tabs. You may see more tabs here if you have one or more extensions installed.

:::

  Here's a brief overview of each tab:

1. **View Settings** - General settings that apply to the entire View
2. **Multiple Entries** - Settings related to the Multiple Entries Layout
3. **Single Entry** - Change the Single Entry page title and other options
4. **Edit Entry** - Allow users to edit entries they create and change the button text for the Update, Delete and Cancel buttons
5. **Delete Entry** - Allow users to delete entries they created
6. **Filter &amp; Sort** - Select various options for sorting entries including restricting by date
7. **Permissions** - Configure user permissions related to your View such as allowing users to download data
