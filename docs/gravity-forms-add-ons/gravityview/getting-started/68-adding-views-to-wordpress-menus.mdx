# Adding Views to WordPress menus



### Go to Appearance &gt; Menus

![](https://www.gravitykit.com/wp-content/uploads/2018/01/go-to-appearance-gt-menus.png?1414452611)

### If Views are not visible in the sidebar

![](https://www.gravitykit.com/wp-content/uploads/2018/01/if-views-are-not-visible-in-the-sidebar.png?1414452612)

### Click "Screen Options" tab

Located in the top-right of your screen.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-screen-options-tab.png?1414452613)

### Check the Views checkbox

![](https://www.gravitykit.com/wp-content/uploads/2018/01/check-the-views-checkbox.png?1414452613)

### Click on the new Views option

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-on-the-new-views-option.png?1414452614)

### Check the Views you want to add to your menu and click "Add to Menu"

![](https://www.gravitykit.com/wp-content/uploads/2018/01/check-the-views-you-want-to-add-to-your-menu-and-click-add-to-menu-.png?1414452615)

### The Views are now added as links in your menu

![](https://www.gravitykit.com/wp-content/uploads/2018/01/the-views-are-now-added-as-links-in-your-menu.png?1414452616)
