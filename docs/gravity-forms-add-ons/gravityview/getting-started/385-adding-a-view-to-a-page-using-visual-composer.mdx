# Adding a View to a page using Visual Composer

If you are using [Visual Composer](https://vc.wpbakery.com/) to create your website pages and want to include a View in one of them, just add a Text Block to your page and add the [\[gravityview\]](https://docs.gravitykit.com/article/73-using-the-shortcode) shortcode inside.
