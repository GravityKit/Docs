# How to enable logging and access GravityView logs



### 1. Go to the Gravity Forms Settings page

Click on the Forms menu, then click on the Settings submenu link.

![](https://www.gravitykit.com/wp-content/uploads/2018/04/go-to-the-gravity-forms-settings-page.png)

### 2. Under General Settings, turn on "Logging"

![](https://www.gravitykit.com/wp-content/uploads/2018/04/under-general-settings-turn-on-logging-.png)

### 3. Click "Save Settings" to update Gravity Forms settings

![](https://www.gravitykit.com/wp-content/uploads/2018/04/click-save-settings-to-update-gravity-forms-settings.png)

Once you save the settings, you will see a new Logging link.

### 4. Click the Logging tab

![](https://www.gravitykit.com/wp-content/uploads/2018/04/click-the-logging-tab.png)

### 5. Set GravityView to "Log all messages" and save the form.

![](https://www.gravitykit.com/wp-content/uploads/2018/04/set-gravityview-to-log-all-messages-and-save-the-form.png)

### 6. Use GravityView as normal

If you're experiencing a specific issue, perform the steps necessary to replicate the problem.

### 7. Return to the Gravity Forms Logging page

There should be a new link next to the "Log all messages" dropdown with the text "view log"

![](https://www.gravitykit.com/wp-content/uploads/2018/04/return-to-the-gravity-forms-logging-page.png)

### 8. Right-click the view log link (or hold down Control while clicking)

Select the option similar to the following options:

- In Safari: "Download Linked File"
- In Firefox: "Save Link As…"
- In Chrome: "Save Link As…"

The text may vary slightly depending on your browser and operating system.

![](https://www.gravitykit.com/wp-content/uploads/2018/04/right-click-the-view-log-link-or-hold-down-control-while-clicking-.png)

### 9. Send us the downloaded log file

Once we have the log file, we'll be able to assist you better.

![](https://www.gravitykit.com/wp-content/uploads/2018/04/send-us-the-downloaded-log-file.png)
