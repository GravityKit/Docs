# How to hide the GravityKit menu

![Screenshot showing the GravityKit menu on the WordPress Dashboard](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/64515b9f40ac037159fb1e71/file-UMJOfAm3hF.png)

If, for some reason, you need to forcefully hide the GravityKit menu without having to mess with [capabilities](https://docs.gravitykit.com/article/311-gravityview-capabilities), then you can use the filter below:

```php
add_filter( 'gk/foundation/admin-menu/submenus', function () {
	return [];
}, 100 );
```

That will hide the menu for everyone, **including yourself!**
