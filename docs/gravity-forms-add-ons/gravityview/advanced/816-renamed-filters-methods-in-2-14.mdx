# Renamed Filters & Methods in 2.14

In GravityView 2.14, we renamed all instances of `blacklist` to `blocklist` and `whitelist` to `allowlist`. All methods and filters have been deprecated using `apply_filters_deprecated()` and `_deprecated_function()`.


:::caution

**We may remove these deprecated filters and functions after one year. Please update your code!**

:::

 ## Changed method names:

| **Old Method Name** | **New Method Name** |
|---|---|
| `GravityView_Cache::blacklist_add()` | `GravityView_Cache::blocklist_add()` |
| `GravityView_Cache::blacklist_remove()` | `GravityView_Cache::blocklist_remove()` |
| `GravityView_Cache::in_blacklist()` | `GravityView_Cache::in_blocklist()` |
| `GravityView_Admin_Views::default_field_blocklist()` | `GravityView_Admin_Views::default_field_blacklist()` |
| `GravityView_Edit_Entry::get_field_blacklist()` | `GravityView_Edit_Entry::get_field_blocklist()` |
| `GravityView_Edit_Entry::modify_field_blacklist()` | `GravityView_Edit_Entry::modify_field_blacklist()` |
| `GravityView_Field_Calculation::blacklist_field_types()` | `GravityView_Field_Calculation::blocklist_field_types()` |
| `GravityView_Field_Product::edit_entry_field_blacklist()` | `GravityView_Field_Product::edit_entry_field_blocklist()` |
| `GravityView_Field_Total::add_to_blacklist()` | `GravityView_Field_Product::add_to_blocklist()` |
| `GravityView_Plugin_Hooks_Gravity_Forms_Coupon::edit_entry_field_blacklist()` | `GravityView_Plugin_Hooks_Gravity_Forms_Coupon::edit_entry_field_blocklist()` |
| `GravityView_Admin_Views::default_field_blacklist()` | `GravityView_Admin_Views::default_field_blocklist()` |

## Changed filter names:

| **Old Filter Name** | **New Filter Name** |
|---|---|
| `gravityview_blacklist_field_types` | `gravityview_blocklist_field_types` |
| `gravityview/sortable/field_blacklist` | `gravityview/sortable/field_blocklist` |
| `gravityview/widget/hide_until_searched/whitelist` | `gravityview/widget/hide_until_searched/allowlist` |
| `gravityview/search/searchable_fields/whitelist` | `gravityview/search/searchable_fields/allowlist` |
| `gravityview/search/operator_whitelist` | `gravityview/search/operator_allowlist` |
