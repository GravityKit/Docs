# The text inside a column is going out of the screen

Here's how to fix a common DataTables layout issue:

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5cf6e91804286333a2642c8f/file-kA6Eb4Wn49.png)

Just [add the following CSS to your theme's stylesheet](https://docs.gravitykit.com/article/246-adding-custom-css-to-your-website):

```
.gv-datatables-container tr.child p {
   white-space: pre-line;
}
```

That will make the text line break like this:

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5cf6e9752c7d3a3837131c0d/file-KcZzVjro4I.png)
