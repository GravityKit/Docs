# Buttons (DataTables Setting)

![Screenshot of the buttons in DataTables](https://i.gravitykit.com/LqqRCQ+)

Display buttons that allow users to print or export the current results.

### Print

The "Print" button opens a new window for the user, generates a table with a copy of the data from the original table, and triggers the browser's print function so the user can print the current table.

### Column Visibility

![Screenshot of the Column Visibility setting](https://i.gravitykit.com/KDSxaP+)

The Column Visibility button allows users to choose which columns to show or hide. The available columns to the user are determined by the fields added to the View layout tabs.
