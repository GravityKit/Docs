# Multi-sorting columns in DataTables


:::caution

Note: Requires GravityView 2.3 or newer.

:::



Sorting a column in DataTables is as easy as clicking the column header. But what about sorting by more than one column at once?

1. Click the table column header to sort by the first column
2. While holding down the Shift key on your keyboard, click the second column, you want to sort by

For more detailed examples of how multi-column sorting works, please read the [article on GravityView multi-column sorting](https://docs.gravitykit.com/article/570-sorting-by-multiple-columns).
