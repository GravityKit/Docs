# How to Disable the DataTables Search Filter

When using the DataTables Extension, Views are displayed with a search input provided by the [DataTables script](http://datatables.net/). If you are also using the [GravityView Search Bar](https://docs.gravitykit.com/article/189-configuring-the-search-bar), you may not want both of these enabled.

To disable the DataTables search input, download and install this plugin: [GravityView - Disable DataTables Search Filter](https://github.com/katzwebservices/GravityView-Disable-DataTables-Search-Filter/archive/master.zip). Once the plugin is installed and activated, you will no longer see the DataTables search input.
