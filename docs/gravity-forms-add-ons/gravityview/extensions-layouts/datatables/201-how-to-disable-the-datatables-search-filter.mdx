# How to Disable the DataTables Search Filter

When using the DataTables Extension, Views are displayed with a search input (red outline) provided by the [DataTables script](http://datatables.net/). You may not want both of these enabled if you are also using the [GravityView Search Bar](https://docs.gravitykit.com/article/189-configuring-the-search-bar) (blue outline).

![Screenshot of the View showing two search inputs](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/643dc4667133de139103ec09/file-hoPdiIApIS.png)  
To disable the DataTables built-in search input, use the code snippet below:

```php

add_filter( 'gravityview_datatables_js_options', function( $config ) {

 $config['searching'] = false;

 return $config;
}, 20 );
```
