# DataTables: is it possible to exclude fields from printing?

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d9f97ef04286364bc904341/file-LPs4DN21iy.png)

Enabling the Print button on a DataTables View allows you to print the visible entries.


:::info

It is impossible to exclude a field or fields that appear in the View from printing, unfortunately.

:::
