# Removing the striped background from DataTables

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5ad8fcdc0428630750929c9e/file-SNo6RTKpgi.jpg)

If you wish to remove that white-blue striped background from your DataTables Views, then you'll need this custom CSS style:

```css
.gv-container div.dataTables_scrollBody {
 background: none !important;
}
```

Here's how to add that to your website's stylesheet: [Adding custom CSS to your website](https://docs.gravitykit.com/article/246-adding-custom-css-to-your-website)
