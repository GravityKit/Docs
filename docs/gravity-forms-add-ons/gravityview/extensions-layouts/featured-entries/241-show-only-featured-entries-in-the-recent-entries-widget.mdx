# Show only Featured Entries in the Recent Entries widget

*Requires Featured Entries Extension Version 1.1 &amp; GravityView Version 1.7*

![Only show featured entries checkbox screenshot](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/54f949e5e4b034c37ea940ac/file-DHxKMO5sTE.png)

### To show only featured entries in the Recent Entries widget:

1. Modify the [Recent Entries widget](http://docs.gravityview.co/article/223-the-recent-entries-widget)
2. Check the "Only show featured entries" checkbox (as shown above)
3. Save the widget

Once you do that, only entries that are featured will be displayed.
