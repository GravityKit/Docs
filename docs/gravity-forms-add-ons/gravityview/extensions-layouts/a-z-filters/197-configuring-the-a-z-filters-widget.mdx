# Configuring the A-Z Filters Widget


In this article, we’ll show you how to get set up with the A-Z Filters extension.

## Installing A-Z Filters

You can install the A-Z Filters extension from inside WordPress. First, hover over "GravityKit" and click "Products &amp; Licenses".

![The Products & Licenses link underneath the GravityKit menu item](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/63a563439ef7901532ad2a8c/file-7ZM2RnkKU3.png)

Now scroll down to "A-Z Filters" (under "Extensions") and click "Install Now".

![The Install Now button for the GravityView A-Z Filters extension](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/63a56a97f9c92f5c53dd6b77/file-U0zeNKhVW0.png)

You can now add the A-Z filter to your View layouts.

## Adding an A-Z Filter to Your View

The A-Z Filter extension adds a new widget to GravityView - the “A-Z Entry Filter” widget. You can add this new widget to your View by clicking on the “Add Widget” button in the top and bottom widget areas.

![The A-Z Entry Filter widget in the list of GravityView widgets](https://lh4.googleusercontent.com/YJHBgJPfXvT7vc46g-tQ0YrjwcDMaw4vz56kuahFUtFYyrUC7MaPqJ69H3pbq_7RzYs-fI9fm2GKg9YIsaFzKiZoa11Yp12NE1iyOC-O759CjPUFbug8lOvfB9fKNIHcVvaZ97IL)

To change the widget settings, click on the gear icon next to the widget label.

![The gear icon next to the widget label](https://lh4.googleusercontent.com/6-KFkGzHro7MMph-Zl260RIhqnmdV0fdyanToUT3rcIP4CRTlzc4B0qzYuony8qBY0DhYjoZt5AMXmftDUo2pMx885YMGFjyH2osbeY9495Nj5kgRWOIARsQDML675NlvDj8MFgi)

## A-Z Filter Widget Settings

When you open up the A-Z Entry Filter Settings, you’ll see three options:

- **Use this field to filter entries** - Select a field in your View. Entries will be filtered by the first character of this field.
- **Alphabet** - Select the alphabet that should be used (this will depend on the language of the information in your View).
- **Use Uppercase Letters?** - Choose whether you want the A-Z filter links to be capitalized or lowercase.

![The A-Z Filter widget settings](https://lh4.googleusercontent.com/yTeeuXXnSnKdsunoxma11V6WwDg4c21yaVjjXQFk748uO7OwqYkHD_YCBQZorDTnUYVzqteUUERoFwY0QHOE9bpSjMbYTfBfSaY-2p7tmMF-IJXbynjqwdqvI9hyRLZCavzA6lSE)

When you’re done, update your View and check it on the front end.

![A-Z Filter alphabet links above a View on the front end](https://lh5.googleusercontent.com/rNmUXo9IQa9vwQw8BqPyJetNmxMyWxpIKo2xdR9ep3wfg6KwDXag8kfMrjhKq_STbbkJprRx8C3ldXpHdTaXX7OJQYztuNF0wpnIjjuNC-vA1XYfU26UDAceThzsEO_vf19dNuRR)

That's it!
