# Configuring info boxes in GravityView Maps



### In View Settings, click on the Maps tab

![](https://www.gravitykit.com/wp-content/uploads/2018/01/in-view-settings-click-on-the-maps-tab.png?1460016447)

### Check the "Show Info Boxes" checkbox

![](https://www.gravitykit.com/wp-content/uploads/2018/01/check-the-show-info-boxes-checkbox.png?1460016448)

### Configure the Info Box settings

**Title:** This text will be used as a link to the single entry. If left blank, the default link text will be "View Details".

**Content:** You can use Merge Tags, shortcodes, and HTML in the Info Box content. Works well with [\[gv\_entry\_link\]](https://docs.gravitykit.com/article/287-gv-entry-link) and [\[gvlogic\]](https://docs.gravitykit.com/article/252-gvlogic-shortcode) shortcodes.

**Image URL** must be the full URL of an image. Use with Merge Tags from File Upload fields.

**Image Alignment** changes the placement of the image in the Info Box.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/configure-the-info-box-settings.png?1460016449)
