# Signing up for a Google Maps API Key


:::caution

**Notice**: An API Key is **required** for Google Maps. You can read more about [this Google guideline here](https://cloud.google.com/maps-platform/user-guide/).

:::



### 1. Create your Google Maps API

Follow the instructions on Google's Get API Key page. **[Click here to go to the Get API Key page](https://developers.google.com/maps/documentation/geocoding/get-api-key) or watch the video from Google below:

<iframe allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen="" frameborder="0" height="315" src="https://www.youtube.com/embed/2_HZObVbe-g" title="YouTube video player" width="560"> </iframe>

  

## 2. API Configurations Required

Now that you have your API Key, you must set the Application restrictions to "None" to avoid further issues, and restrict your API Key to Geocoding API, Geolocation API and Maps JavaScript API.   
![Screenshot of the Application restrictions settings on Google's website](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/60cbb12a61c60c534bd67142/file-nsbNbKRGol.png)

##  3. Adding the API Key to GravityView's Settings 

Back on your site, visit the GravityView Settings page:

![The Settings link under Views in the WordPress sidebar](https://www.gravitykit.com/wp-content/uploads/2018/08/back-on-your-site-visit-the-gravityview-settings-page.png)

Paste the API key into your GravityView Settings. If you don't see the setting, it's because the GravityView Maps plugin isn't activated. Check your Plugins page.

![The Google Maps API key input box on the GravityView Settings page](https://www.gravitykit.com/wp-content/uploads/2018/08/paste-the-api-key-into-your-gravityview-settings.png)

Update your GravityView settings.

![The Update button](https://www.gravitykit.com/wp-content/uploads/2018/08/update-your-gravityview-settings-.png)

You're all set now with your Google Maps API key!
