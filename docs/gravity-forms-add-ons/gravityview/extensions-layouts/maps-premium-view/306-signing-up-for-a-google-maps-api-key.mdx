# Signing up for a Google Maps API Key


:::caution

**Notice**: An API Key is **required** for Google Maps. You can read more about [this Google guideline here](https://cloud.google.com/maps-platform/user-guide/).

:::



### 1. Create your Google Maps API

Follow the instructions on Google's Get API Key page. **[Click here to go to the Get API Key page](https://developers.google.com/maps/documentation/geocoding/get-api-key) or watch the video from Google below:

import YouTube from 'react-youtube';

<YouTube videoId="2_HZObVbe-g" />

  

## 2. API Configurations Required

Now that you have your API Key, you must set the Application restrictions to "None" to avoid further issues, and restrict your API Key to Geocoding API, Geolocation API and Maps JavaScript API.   
![Screenshot of the Application restrictions settings on Google's website](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/60cbb12a61c60c534bd67142/file-nsbNbKRGol.png)

##  3. Adding the API Key to GravityKit's Settings page

![Screenshot showing GravityKit's Settings page](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/640a38df9a16bd7e6b0b2840/file-7EZOTFT7hI.png)

You're all set now with your Google Maps API key!
