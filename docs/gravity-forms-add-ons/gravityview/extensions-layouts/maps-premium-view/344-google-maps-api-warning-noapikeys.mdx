# Error: Google Maps API warning: NoApiKeys

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/640a35bdcde27207000227d9/file-8OQTzQLdQv.png)

If you see a warning in your browser's Javascript console that says `Google Maps JavaScript API warning: NoApiKeys https://developers.google.com/maps/documentation/javascript/error-messages#no-api-keys` the reason is that you haven't configured your Google Maps API keys or there's a conflict with another plugin/theme trying to load the Google Maps script at the same time.

There are two ways to fix this problem:

### 1) Add your Google Maps API key in the GravityKit's Settings screen

![Screenshot showing GravityKit's Settings page to input the Google Maps API Key](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/640a374c3387cb3e147b74b5/file-MTjwaiexCZ.png)

### 2) Add your Google Maps API key on the settings page of other plugins or your currently active theme

Read more about that here: [Error: This page can't load Google Maps correctly](https://docs.gravitykit.com/article/870-this-page-cant-load-google-maps-correctly)
