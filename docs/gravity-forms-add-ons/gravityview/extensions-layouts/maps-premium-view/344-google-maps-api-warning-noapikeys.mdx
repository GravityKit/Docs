# I see a warning "Google Maps API warning: NoApiKeys"


:::caution

**Notice**: Beginning on June 11, 2018, an API Key is now **required** for every map project using the Google Maps library. You can read more about [this Google guideline here](https://cloud.google.com/maps-platform/user-guide/).

:::



![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/570545d0903360288a7807fb/file-QCFxL8DMPW.png)

If you see a warning in your browser's Javascript console that says "Google Maps API warning: NoApiKeys [https://developers.google.com/maps/documentation/javascript/error-messages#no-api-keys](https://developers.google.com/maps/documentation/javascript/error-messages#no-api-keys)", the reason is that you haven't configured your Google Maps API keys yet.

Please enter your Google Maps API key in the GravityView Settings screen. Learn more in this article: [Signing up for a Google Maps API key](https://docs.gravitykit.com/article/306-signing-up-for-a-google-maps-api-key).
