# Signing up for a Bing Maps API Key



### 1. Sign in to the Bing Maps Dev Center

**Go to the [Bing Maps Dev Center](https://www.bingmapsportal.com)**.

You will need to have a Microsoft Account. If you don't have one, create one by following the steps on the page.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/sign-in-to-the-bing-maps-dev-center.png?1443204416)

### 2. Under "My account" menu, click "Create or view keys"

![](https://www.gravitykit.com/wp-content/uploads/2018/01/under-my-account-menu-click-create-or-view-keys-.png?1443204416)

### 3. Click the link to create a new key

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-link-to-create-a-new-key.png?1443204416)

### 4. Create a key called GravityView Maps

Set the "Application type" to Public website. The Application URL is optional.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/create-a-key-called-gravityview-maps.png?1443204417)

### 5. Select the Key and copy it

You can also click the "Copy key" link, which might work, depending on your browser.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/select-the-key-and-copy-it.png?1443204418)

### 6. Visit the GravityView Settings page

Navigate to Views &gt; Settings on your WordPress site.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/visit-the-gravityview-settings-page.png?1443204419)

### 7. Paste the API key into your GravityView Settings

If you don't see the setting, it's because the GravityView Maps plugin isn't activated. Check your Plugins page.

You're all set now with your Bing Maps API key.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/paste-the-api-key-into-your-gravityview-settings.png?1443204419)
