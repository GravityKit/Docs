# How to show markers from multiple address fields on a map



### To display markers for a multiple address fields, click on a field name

![](https://www.gravitykit.com/wp-content/uploads/2018/10/to-display-markers-for-a-multiple-address-fields-click-on-a-field-name.png)

### Hold down the Shift key on your keyboard, then click on the last address field you want to select

![](https://www.gravitykit.com/wp-content/uploads/2018/10/hold-down-the-shift-key-on-your-keyboard-then-click-on-the-last-address-field-you-want-to-select.png)

### A marker will now be shown for each of the address fields
