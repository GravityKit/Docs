# Adding your Google Maps API Key into Premium Addons for Elementor

## 1) Click on the Integrations page of the Premium Addons for Elementor menu:

![Screenshot of the Premium Addons for Elementor menu](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/63b88354e8beff1be5c6696d/file-qmCxfx67GE.png)

### 2) Find the "Google Maps API Key" field and paste your key, changes will be automatically saved. That's it!

  
![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/63b88408e8beff1be5c6696f/file-E6nj9Caqns.png)
