# Adding your Google Maps API Key into Ultimate Addons for Elementor

## 1) Find the Ultimate Addons for Elementor settings page:

![Screenshot of the Ultimate Addons for Elementor](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/63054552c1ba4b3dfdae2636/file-pcWIR1wKpB.png)

### 2) Find "Google Map", and click on Settings:

![Screenshot of the Ultimate Addons for Elementor](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/630545a9549d66420b99a103/file-wVt8P9wrAc.png)

### 3) Add your Google Maps API Key and click Save Changes:

![Screenshot of the Ultimate Addons for Elementor](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6305463c549d66420b99a108/file-n9ByB4tXlh.png)
