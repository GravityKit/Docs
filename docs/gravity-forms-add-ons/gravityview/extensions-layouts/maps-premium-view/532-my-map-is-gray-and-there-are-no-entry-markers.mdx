# My map is gray and there are no entry markers

The problem here is that your entries haven't been Geolocated yet, so you must now go to Google's Developer Console &gt; [Credentials](https://console.developers.google.com/apis/credentials) &gt; On the right side, look for the list of your API keys.

  
Click on your API:  
  
![](https://d33v4339jhl8k0.cloudfront.net/inline/24990/52223100c9ebfc803ca16db92808f5f50c90a889/c40f2137b5d9cd46a36553d9036ac02be45e13ee/b30b1bcea39085d1dab2a30d82b09331.png)  
  
Once inside, you'll see this page:  
  
![](https://d33v4339jhl8k0.cloudfront.net/inline/24990/3d6d3256c305e1e5401bfbe55eaebb0bc96ad674/f5a1d7768d4cdd362113a216aea0451bd12bc1dd/google-api-1.png)  
  
Then, click on API restrictions, and follow these two steps below:  
  
![](https://d33v4339jhl8k0.cloudfront.net/inline/24990/a91122d97f8734a4b2ecf17514a5695372431071/a934bc937ef3c9027a5ff2c8f9b9c61c53a598e8/gv-api-2.png)  
  
It should look like this:  
  
![](https://d33v4339jhl8k0.cloudfront.net/inline/24990/3fec01392da6480fb31174b4d0944b2a723411b1/e8dd29617d2f328ec026910fb3a1e133b447338a/gv-api-3.png)  
  
After doing this, go back to each one of your form entries, they should appear with this message:  
  
![](https://d33v4339jhl8k0.cloudfront.net/inline/24990/7d12d140866774a33bce3acbd716932c3b45d18a/8710fa9e98578694ab7beb9658515844a3814028/gv-not-geolocated-1.png)  
  
Click on "Geocode This Address":  
  
![](https://d33v4339jhl8k0.cloudfront.net/inline/24990/ee1a3d309d78e8a3dffcc525b5557d4467688421/d92abc1720a70beda64ffb347d9755305ed8da43/gv-not-geolocated-2.png)  
  
That should make the entry shows up on the map.
