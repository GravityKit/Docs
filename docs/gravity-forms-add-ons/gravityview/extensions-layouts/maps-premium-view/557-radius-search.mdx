# Can we search for a business near the user?

Searching entries near the user's current location (also called Radius Search) is not yet possible with our Maps extension.


:::info

This functionality is finally coming. Make sure to [subscribe to our newsletter](https://www.gravitykit.com/newsletter/) to be informed of its release.

:::
