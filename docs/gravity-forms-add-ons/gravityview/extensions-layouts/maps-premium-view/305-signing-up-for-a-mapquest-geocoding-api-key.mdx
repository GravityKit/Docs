# Signing up for a MapQuest Geocoding API Key



### 1. Click "Get Started" in the Free plan column

**Visit the [MapQuest Developer Plans](<https://developer.mapquest.com/plans >) page to get started.**

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-get-started-in-the-free-plan-column.png?1443201387)

### 2. Fill out your information

![](https://www.gravitykit.com/wp-content/uploads/2018/01/fill-out-your-information.png?1443201390)

### 3. Click "Keys &amp; Reporting", then "My Application"

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-keys-amp-reporting-then-my-application-.png?1443201392)

### 4. Click "Edit"

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-edit-.png?1443201392)

### 5. Rename the App Name

![](https://www.gravitykit.com/wp-content/uploads/2018/01/rename-the-app-name.png?1443201393)

This is optional, but recommended so that it's clear in the future what the API Key is being used for.

### 6. Click GravityView Maps and copy the Consumer Key

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-gravityview-maps-and-copy-the-consumer-key.png?1443201394)

You'll be pasting the key into GravityView Maps settings in the next step.

### 7. Go to the GravityView Settings page

![](https://www.gravitykit.com/wp-content/uploads/2018/01/go-to-the-gravityview-settings-page.png?1443201395)

### 8. Paste the key into your GravityView Settings screen under "MapQuest Geocoding API Key"

![](https://www.gravitykit.com/wp-content/uploads/2018/01/paste-the-key-into-your-gravityview-settings-screen-under-mapquest-geocoding-api-key-.png?1443201396)
