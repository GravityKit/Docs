# Sorting results by Field value



### 1. Edit the View you want to sort

On the All Views page, click on the View you wish to sort.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/edit-the-view-you-want-to-sort.png?1432159114)

### 2. Find the Filter &amp; Sort tab of the Settings box

The Settings box is under the View layouts tabs. If you can't see it, [please follow this how-to first](https://docs.gravitykit.com/article/87-the-settings-box-is-hidden).

![Screenshot of the Settings metabox with the Filter & Sort tab active.](https://i.gravitykit.com/MsVffK+)

### 3. Choose what field to sort by

This field will be used to sort the results alphabetically. If the field is numeric, it will be sorted by value.

![Screenshot of 'Sort by field' setting dropdown open](https://i.gravitykit.com/TKFxGv+)

### 4. Choose the sort direction

"ASC" means ascending; A first and Z last.

"DESC" means descending: Z first and A last.

"Random" means items will be randomly sorted.

![Sort direction dropdown open ](https://i.gravitykit.com/QxHUso+)

### 5. Save the View

Once you have updated your sorting settings, save your changes.

![Update button in Publish box](https://i.gravitykit.com/4VD173+)
