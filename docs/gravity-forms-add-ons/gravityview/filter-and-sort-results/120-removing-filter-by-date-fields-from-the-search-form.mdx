# Removing "Filter by Date" fields from the search form


:::info

This article is valid for GravityView versions up to 1.1.6. For versions 1.2+, check [this tutorial about how to set up the Search Bar](https://docs.gravitykit.com/article/189-configuring-the-search-bar).

:::



## Click the "Configure Field Settings" gear next to the Show Search Bar widget

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-configure-field-settings-gear-next-to-the-show-search-bar-widget.png?1407794781)

## Uncheck the "Show date filters" checkbox

![](https://www.gravitykit.com/wp-content/uploads/2018/01/uncheck-the-show-date-filters-checkbox.png?1407794781)

## Add or Update the View

Don't forget to save your changes!

## The date fields will no longer appear in your search form
