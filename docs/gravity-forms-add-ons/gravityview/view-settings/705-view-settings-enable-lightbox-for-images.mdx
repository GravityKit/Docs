# View Settings: Enable Lightbox for Images

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5ff70b0466df373cab7071da/file-ifBRYvMg0j.png)  
If enabled, images will open full-size in a "lightbox". A lightbox displays images and videos by filling the screen and dimming out the rest of the web page.

If disabled, clicking the image will not enlarge the image.
