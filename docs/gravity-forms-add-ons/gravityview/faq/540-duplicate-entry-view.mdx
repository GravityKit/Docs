# Can I duplicate an entry from a View?

Yes, that's now possible since GravityView Version 2.5.

In Edit View, click the Add Field button and the Duplicate Entry field.

That will add a link to duplicate an entry from a View. [Read this article for a full step-by-step guide](https://docs.gravitykit.com/article/663-duplicate-an-entry).
