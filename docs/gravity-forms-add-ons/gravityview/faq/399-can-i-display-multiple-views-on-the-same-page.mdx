# Can I display multiple Views on the same page?

Yes, you can!

If you are embedding a View into a page, then it is just a matter of adding the [\[gravityview\]](https://docs.gravitykit.com/article/73-using-the-shortcode) shortcode for each View in the WordPress editor like this:

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/585d9b2cc697915403a0692d/file-Sv6koLW7g9.jpg)

Some problems might happen if you add two different Views using DataTables for both of them, so please make sure to try it first.
