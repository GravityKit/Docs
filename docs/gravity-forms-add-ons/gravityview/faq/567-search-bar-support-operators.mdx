# Does the Search Bar support operators like OR or AND?

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5f73ce6ecff47e00168fb979/file-oORbGluIUl.png)

Yes, when accessing the Search Bar configuration modal the Search Mode section that appears when you add two or more fields shows up to reveal the options to choose the search operators for the fields added.

Unfortunately, it's not yet possible to add multiple conditions.
