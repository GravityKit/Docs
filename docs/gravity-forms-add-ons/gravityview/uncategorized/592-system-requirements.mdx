# GravityView System Requirements

GravityView recommends the same [system requirements as Gravity Forms](https://docs.gravityforms.com/gravity-form-system-requirements/), including

- PHP 5.6 or newer
- MySQL version 5.5+
- The latest version of WordPress

GravityView supports newer versions of PHP, including PHP 8.1.


:::caution

Please note: GravityView does not provide customer support for websites running on [IIS](https://www.iis.net) servers.

:::
