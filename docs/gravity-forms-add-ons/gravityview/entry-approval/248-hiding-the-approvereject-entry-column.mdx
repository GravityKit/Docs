# Hiding the Approve/Reject Entry column

![Screenshot of the entries page of Gravity Forms](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/6063a988207e3b1188e21bda/file-OVXMGnBQm1.png)

If you only want to display the Approve/Reject Entry column when there are Views connected to the form, you can use the following code [in your functions.php file](https://docs.gravitykit.com/article/210-where-to-put-code-samples):

```
add_filter('gravityview/approve_entries/hide-if-no-connections', '__return_true');
```
