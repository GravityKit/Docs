# How to display the total number of Approved entries

You can display the total number of Approved entries on your website with the [Advanced Filter extension](https://www.gravitykit.com/extensions/advanced-filter/) and a Custom Content widget. To set this up:

### Set Your View to Show Approved Entries Only

With the [Advanced Filter extension](https://www.gravitykit.com/extensions/advanced-filter/) enabled, edit your View and click on the "Show only approved entries" checkbox in the View Settings:

![Enabling "Show only approved entries" under View Settings](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d13c2842c7d3a6ebd2295e7/file-naWyqfurPs.png)

### Add a Custom Content Widget

Choose the area where you want to display the approved entries then click the Add Widget button:

![Clicking the Add Widget button to add a widget below the entries](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d13c42a04286305cb87be09/file-c7MwB6M1OP.png)

Select the Custom Content widget from the list of available widgets:

![Clicking on Custom Content from the list of available widgets](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d13c5972c7d3a6ebd22961d/file-ogJakKiuJ4.png)

### Configure the Widget to Show the Total Number of Approved Entries

Click the gear icon to configure the new widget:

![Clicking the gear icon to customize the widget](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d13c7882c7d3a6ebd22964c/file-5acbm4UPyB.png)

In the text box, type the text you wish to display and the shortcode used to show the number of approved entries, `[gravityview detail="total_entries"]`, then click the Close button to save.

![Entering the text and shortcode to display the total number of approved entries](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d13c90004286305cb87be5d/file-vbngjEE3KS.png)

### Save or Update your View

![Clicking the Update button to save the modified View](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d0bde900428637fd7c53c0c/file-XD4ISXQkvK.png)

### The Total Number of Approved Entries Will Now Be Visible

![Total number of approved entries now showing on the website](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d13ce2104286305cb87be9d/file-ZhZnS0VuwX.png)
