# GravityEdit: How to Enable for GravityView Views



### Edit the View for which you want to enable Inline Edit

![](https://www.gravitykit.com/wp-content/uploads/2018/01/edit-the-view-for-which-you-want-to-enable-inline-edit.png?1500399893)

Go to Views &gt; All Views, and then click the View itself.

### Find the View Settings box, and check the "Enable Inline Edit" checkbox

![](https://www.gravitykit.com/wp-content/uploads/2018/01/find-the-view-settings-box-and-check-the-enable-inline-edit-checkbox.png?1500399894)

Don't see the View settings box? Typically, it's near the bottom of the page. If you still can't find it, [here's an article to help](https://docs.gravitykit.com/article/87-the-settings-box-is-hidden).

### Update the View

![](https://www.gravitykit.com/wp-content/uploads/2018/01/update-the-view.png?1500399894)

### Check out the View on your site

![](https://www.gravitykit.com/wp-content/uploads/2018/01/check-out-the-view-on-your-site.png?1500399894)

### You should now see the Inline Edit link above your View!

![](https://www.gravitykit.com/wp-content/uploads/2018/01/you-should-now-see-the-inline-edit-link-above-your-view-.png?1500399895)

If you don't see the link, that's not your fault; [let us know!](https://www.gravitykit.com/support/)
