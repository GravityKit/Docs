# How to enable duration calculations in Gravity Forms with GravityMath



### Click the "Single Line Text" button to add a text field

![](https://www.gravitykit.com/wp-content/uploads/2021/02/click-the-single-line-text-button-to-add-a-text-field.png)

### Click the new field to access the field settings

![](https://www.gravitykit.com/wp-content/uploads/2021/02/click-the-new-field-to-access-the-field-settings.png)

### Check the box titled Input Mask

![](https://www.gravitykit.com/wp-content/uploads/2021/02/check-the-box-titled-input-mask.png)

### Choose a Duration format from the "Select a Mask" dropdown

![](https://www.gravitykit.com/wp-content/uploads/2021/02/choose-a-duration-format-from-the-select-a-mask-dropdown.png)

"Duration (MM:SS)" will allow users to enter values in minute/second format.

"Duration (HH:MM:SS)" will allow users to enter values in hour/minute/second format.

### Save the form

![](https://www.gravitykit.com/wp-content/uploads/2021/02/save-the-form.png)
