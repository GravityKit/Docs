# Import CSV to Gravity Forms with Import Entries 2.0



## Start by going to the menu called "Import/Export" in Gravity Forms' "Forms" menu

![Start by going to menu called "Import/Export" in Gravity Forms' "Forms" menu](https://www.gravitykit.com/wp-content/uploads/2019/01/start-by-going-to-menu-called-import-export-in-gravity-forms-forms-menu.png)

---

## Click Import Entries

![Click Import Entries](https://www.gravitykit.com/wp-content/uploads/2019/01/click-import-entries.png)

---

## Drag a CSV file to the box, or click the box to select a CSV from your computer to upload.

![Drag a CSV file to the box, or click the box to select a CSV from your computer to upload.](https://www.gravitykit.com/wp-content/uploads/2019/01/drag-a-csv-file-to-the-box-or-click-the-box-to-select-a-csv-from-your-computer-to-upload.png)

---

## Choose whether to import entries into an existing form, or to create a new form

![Choose whether to import entries into an existing form, or to create a new form](https://www.gravitykit.com/wp-content/uploads/2019/01/choose-whether-to-import-entries-into-an-existing-form-or-to-create-a-new-form.png)

---

## If you create a new form, the columns in the CSV will be turned into form fields in a new form

![If you creatae a new form, the columns in the CSV will be turned into form fields in a new form](https://www.gravitykit.com/wp-content/uploads/2019/01/if-you-creatae-a-new-form-the-columns-in-the-csv-will-be-turned-into-form-fields-in-a-new-form.png)

---

## If you choose to import into an existing form, click on the form you want to import into

![If you choose to import into an existing form, click on the form you want to import into](https://www.gravitykit.com/wp-content/uploads/2019/01/if-you-choose-to-import-into-an-existing-form-click-on-the-form-you-want-to-import-into.png)

### If you have many forms, you can type to search for the form you want to import to

![If you have many forms, you can type to search for the form you want to import to](https://www.gravitykit.com/wp-content/uploads/2019/01/if-you-have-many-forms-you-can-type-to-search-for-the-form-you-want-to-import-to.png)

---

## To modify the CSV file or the form being imported into, you can click the links at the top at any time

![To modify the CSV file or the form being imported into, you can click the links at the top at any time](https://www.gravitykit.com/wp-content/uploads/2019/01/to-modify-the-csv-file-or-the-form-being-imported-into-you-can-click-the-links-at-the-top-at-any-ti.png)

You can also press the back button in your browser.

---

## A preview of the uploaded file will be shown.

![A preview of the uploaded file will be shown.](https://www.gravitykit.com/wp-content/uploads/2019/01/a-preview-of-the-uploaded-file-will-be-shown.png)

---

## Choose which field the column data should be imported into

![Choose which field the column data should be imported into](https://www.gravitykit.com/wp-content/uploads/2019/01/choose-which-field-the-column-data-should-be-imported-into.png)

---

## If a field does not exist, you can create one by selecting "Add Form Field"

![If a field does not exist, you can create one by selecting "Add Form Field"](https://www.gravitykit.com/wp-content/uploads/2019/01/if-a-field-does-not-exist-you-can-create-one-by-selecting-add-form-field-.png)

### When adding a form field, you will be prompted to define the field label and the type of field

![When adding a form field, you will be prompted to define the field label and the type of field](https://www.gravitykit.com/wp-content/uploads/2019/01/when-adding-a-form-field-you-will-be-prompted-to-define-the-field-label-and-the-type-of-field.png)

---

## To skip a column and not import it, select "Do Not Import" from the field dropdown

![To skip a column and not import it, select "Do Not Import" from the field dropdown](https://www.gravitykit.com/wp-content/uploads/2019/01/to-skip-a-column-and-not-import-it-select-do-not-import-from-the-field-dropdown.png)

### When a column is set to Do Not Import, the data preview will be grayed out.

![When a column is set to Do Not Import, the data preview will be grayed out.](https://www.gravitykit.com/wp-content/uploads/2019/01/when-a-column-is-set-to-do-not-import-the-data-preview-will-be-grayed-out.png)

---

## You can limit what is imported by enabling Conditional Import and defining rules

![You can limit what is imported by enabling Conditional Import and defining rules](https://www.gravitykit.com/wp-content/uploads/2019/01/you-can-limit-what-is-imported-by-enabling-conditional-import-and-defining-rules.png)

You can add multiple rules at once by clicking the plus button to the right of each rule.

---

## Click Continue With Import to begin the import process

![Click Continue With Import to begin the import process](https://www.gravitykit.com/wp-content/uploads/2019/01/click-continue-with-import-to-begin-the-import-process.png)

---

## You will see the progress of the import on the screen.

![You will see the progress of the import on the screen.](https://www.gravitykit.com/wp-content/uploads/2019/01/you-will-see-the-progress-of-the-import-on-the-screen.png)

---

## When the import is completed, you can start a new one or view the entries.

![When the import is completed, you can start a new one or view the entries.](https://www.gravitykit.com/wp-content/uploads/2019/01/when-the-import-is-completed-you-can-start-a-new-one-or-view-the-entries.png)
