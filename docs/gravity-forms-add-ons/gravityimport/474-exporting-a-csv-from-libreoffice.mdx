# Exporting a CSV from LibreOffice


:::caution

Gravity Forms does not include entry import functionality. GravityImport is the best way to import data into Gravity Forms from a CSV. Get it today, and import your entries fast!   
[Buy it now!](https://www.gravitykit.com/extensions/gravity-forms-entry-importer/)

:::



[LibreOffice](https://www.libreoffice.org) is an open-source, completely free office suite. If you want to export your entries from LibreOffice into Gravity Forms, this guide is for you!

### Under *File,* Select *Save As (*Not *<u>Export!)</u>*

### ![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b717cbe0428631d7a89e8b9/file-16BTwM14hX.png)Then, Select *CSV* from the Drop-down Menu

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b717d0e2c7d3a03f89d9fa9/file-VEslpzBfYr.png)

### On the Settings Page, Select *UTF-16*

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b717d1d0428631d7a89e8bf/file-8OsxDjZRaa.png)

Finally, export your file. That's it!
