# Exporting a CSV from Google Sheets


:::caution

Gravity Forms does not include entry import functionality. GravityImport is the best way to import data into Gravity Forms from a CSV. [Get it today, and import your entries fast!](https://www.gravitykit.com/extensions/gravity-forms-entry-importer/)

:::



Google Sheets is the spreadsheet program included in the Google Docs Suite. If you want to export your entries from Google Sheets and import them into Gravity Forms, this guide is for you!

### Under *File,* Select *Download as*

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b717a312c7d3a03f89d9f81/file-goO5KEU4Kr.png)

### Then, Click on *Comma-Separated Values (.csv, current sheet)*

 *![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b717a4f2c7d3a03f89d9f83/file-tNBqZ3vHYW.png)*

And that's it! Your file will be downloaded to your computer.
