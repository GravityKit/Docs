# How to trigger notifications for imported entries


:::caution

The new version 2 of our [Gravity Forms Import Entries](https://www.gravitykit.com/extensions/gravity-forms-entry-importer/) plugin can now trigger notifications during the import phase:

:::



![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5ec711952c7d3a5ea54b8dc7/file-YRV45Ebgnn.png)

## How to trigger notifications after the entries have been imported

#### 1) View the Entries for the form you just imported into

To get to this page, go to Forms &gt; Forms, then hover over the form you are interested in.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/view-the-entries-for-the-form-you-just-imported-into.png?1430449537)

#### 2) Check the checkbox in the top-left of the table to select all visible entries

![](https://www.gravitykit.com/wp-content/uploads/2018/01/check-the-checkbox-in-the-top-left-of-the-table-to-select-all-visible-entries.png?1430449538)

#### 3) To select all entries, click the "Select all entries" link

If you don't see this link, you have already selected all available entries.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/to-select-all-entries-click-the-select-all-entries-link.png?1430449539)

#### 4) Select "Resend Notifications" from the "Bulk action" dropdown

You'll find the dropdown above the form entries table.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/select-resend-notifications-from-the-bulk-action-dropdown.png?1430449539)

#### 5) Click the Apply button

This will process resending notifications for the entries.

![](https://www.gravitykit.com/wp-content/uploads/2018/01/click-the-apply-button.png?1430449540)
