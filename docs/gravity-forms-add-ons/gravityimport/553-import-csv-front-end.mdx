# Can I import a CSV file on the front-end of my website?

Sorry, that's currently not possible to be achieved with our [GravityImport](https://www.gravitykit.com/extensions/gravity-forms-entry-importer/) plugin.

You can only import entries on the back end (Dashboard) of your website.
