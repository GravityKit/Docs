# Error: Server returned an empty response

If you experience the following error message:


:::danger

Server returned an empty response.

:::

  This usually means you have a caching issue. It could be related to a caching plugin installed on your website or even with CloudFlare (if your website uses it).

If you have a cache plugin installed, try deactivating it before starting the import process.
