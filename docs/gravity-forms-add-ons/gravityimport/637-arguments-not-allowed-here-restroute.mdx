# Error: Arguments not allowed here: rest_route

If you are using the default WordPress Permalink structure, you may see the following error when using the GravityImport plugin:

![The Arguments not allowed here: rest_route error appearing during an import](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d96726204286364bc8ff5be/file-XA8DViqmJw.png)

If you encounter this error, please go to Settings -&gt; Permalinks from the WordPress Dashboard. Change the WordPress Permalink Settings from the default to any other options, then restart your import.
