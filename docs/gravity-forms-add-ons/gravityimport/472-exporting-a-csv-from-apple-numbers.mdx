# Exporting a CSV from Apple Numbers

*Numbers* is a spreadsheet program developed by Apple which is included in every OS X installation. If you want to export your entries from Numbers into Gravity Forms / GravityView, this guide is for you!

### Under the *File* menu, select *Export To*

![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b7174d60428631d7a89e865/file-nCee9GipmS.png)

### Then, Select *CSV*

### ![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b7177c10428631d7a89e87d/file-ygnsk1DZVb.png)Finally, Click *Next* and Save Your File

**![](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5b7175672c7d3a03f89d9f45/file-x7QIEeAE9h.png)Note:** If you are having issues importing your file, try exporting it as *Unicode (UTF-16).*
