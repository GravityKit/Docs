# Import Option: Skip Field Validation

![Choosing Skip Field Validation during import options configuration](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d77f8912c7d3a7e9ae0fbf3/file-gfx8vKqInw.png)

Enabling the "Skip Field Validation" option allows you to disable validating the data you import. Warnings about phone number formats, URLs that aren't valid, etc. will not appear. Example:

![An example of a data field validation error during import](https://s3.amazonaws.com/helpscout.net/docs/assets/54c58489e4b051242988532c/images/5d77fa842c7d3a7e9ae0fc08/file-2fLioJUJI8.png)

With "Skip Field Validation" enabled, a new entry will be created even if an imported field contains invalid data.

Note: Validation can be disabled during the import, but **Gravity Forms will still sanitize data for certain fields**. For example, Gravity Forms will still strip invalid content like trying to set a number field to contain text.
