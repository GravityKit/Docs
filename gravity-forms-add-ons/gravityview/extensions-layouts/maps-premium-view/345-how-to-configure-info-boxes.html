Configuring info boxes in GravityView Maps

In View Settings, click on the Maps tab​

Check the "Show Info Boxes" checkbox​

Configure the Info Box settings​

Title: This text will be used as a link to the single entry. If left blank, the default link text will be "View Details".

Content: You can use Merge Tags, shortcodes, and HTML in the Info Box content. Works well with [gv_entry_link] and [gvlogic] shortcodes.

Image URL must be the full URL of an image. Use with Merge Tags from File Upload fields.

Image Alignment changes the placement of the image in the Info Box.
